**Coworker Preferences** control how Coworker behaves for you personally — how it responds, which agent it picks up when you `@`-mention nothing in particular, and which tools it can run without asking first.

Open **Settings → Coworker Preferences**.

## What you can tune

- **Default agent** — the agent that answers when you don't `@`-mention a specific one.
- **Response style** — concise vs. detailed, formal vs. casual.
- **Auto-approved tools** — tools Pinkfish is allowed to call without asking for confirmation each time. Paired with [Tool Approvals](/settings/personal#tool-approvals).
- **Saved Actions behavior** — whether Coworker automatically replays a Saved Action when the request semantically matches one, or always asks.
- **Knowledge base defaults** — which knowledge bases are pre-attached to every new conversation.

## Scope

These preferences apply to **your user** only. Org admins set organization-wide defaults under [Organization Settings](/settings/organization).
