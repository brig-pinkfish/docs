**API Credentials** is where organization admins mint, view, and revoke the API keys that external callers use to hit the Pinkfish API.

Open **Settings → API Credentials** (admin only).

## What an API credential does

An API key authenticates requests to:

- **Trigger APIs** — [Trigger an Automation](/api-reference/endpoint/trigger-an-automation), [Webhook Trigger](/api-reference/endpoint/webhook-trigger).
- **Platform API** — [Authentication](/api-reference/platform/authentication), [Making Requests](/api-reference/platform/making-requests).
- **Agent API** — per-agent keys that let you chat with a released agent programmatically.

## Creating a key

1. Click **New API credential**.
2. Give it a name (e.g. "Production webhook — Salesforce").
3. Pick a scope — what resources the key can access.
4. Copy the secret **once** — it's shown in full only at creation time.

## Rotating a key

Keys can't be re-shown after creation. To rotate:

1. Create a new key with the same scope.
2. Swap it into your caller.
3. Revoke the old key.
