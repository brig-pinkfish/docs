# Platform Limits

Pinkfish implements resource limits to ensure optimal performance, reliability, and fair usage across all customers. These limits are designed to maintain platform stability while providing flexibility for your automation needs.

## Plan Overview

<Tabs>
  <Tab title="Starter">
    Free trial with basic features:
    - Single user
    - Basic automations
    - Limited collections
  </Tab>
  <Tab title="Pro">
    Individual professional use:
    - Single user
    - API connections
    - Browser operator
    - Three tiers with increasing resources
  </Tab>
  <Tab title="Team">
    Collaborative team use:
    - Up to 10 users
    - Multiple organizations
    - Advanced connections
    - Three tiers with team features
  </Tab>
  <Tab title="Enterprise">
    Custom organizational solutions:
    - Unlimited users
    - Highest resource limits
    - All advanced features
    - Optional add-on packages
  </Tab>
</Tabs>

| **Organization**              |         |     |      |            |
| Team Members                  | 1       | 1   | 10   | Unlimited  |

| **Connections**               |         |     |      |            |
| Browser Operator Logins       | 1       | 5   | 10   | 100+       |
| Custom API Connections        | 0       | 5   | 10   | 100+       |
| **Collections**               |         |     |      |            |
| Datastore Collections         | 1       | 1   | 10   | 100+       |
| File Storage Collections      | 1       | 1   | 10   | 100+       |
| Vault Collections             | 0       | 0   | 10   | 100+       |

### Storage Capacity

| Collection Type        | Maximum Items       | Notes                                 |
| ---------------------- | :-----------------: | ------------------------------------- |
| Datastore Collections  | 10,000 per collection| Organization-level for Team/Enterprise|
| File Storage Collections| 1,000 per collection | Organization-level for Team/Enterprise|
| Vault Collections      | 50 per collection    | Organization-level for Team/Enterprise|

<Info>
  Enterprise plans can increase limits with add-ons:
  - Browser Plus: +5 concurrent browser sessions
  - Organization Plus: Additional organizations
</Info>

For information on credits and usage-based billing for services that extend beyond these limits, please refer to our [Credits](/essentials/credits) documentation.
