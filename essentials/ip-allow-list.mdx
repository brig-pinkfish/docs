## What is IP Allow Listing?

When your organization restricts outbound traffic (e.g., via firewall rules, VPN egress, or corporate network policies) to only allow connections from specific IP addresses, you need to allowlist Pinkfish's production IPs. Pinkfish uses static IPs for production traffic so you can reliably configure your network.

## Production Static IPs

Allow the following IP addresses in your firewall or VPN egress rules:

- **34.59.142.239**
- **35.192.58.176**

Allow both IPs for redundancy.
