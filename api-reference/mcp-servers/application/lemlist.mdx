Lemlist via official MCP server. Audit campaigns, create and update sequences, and manage leads. Tools are discovered at runtime. The available tools evolve over time — query the MCP server directly for the current list.

## Tools

| Tool | Description |
| --- | --- |
| `get_campaigns` | Retrieve and audit existing campaigns |
| `get_campaign_stats` | Pull open rates, reply rates, and conversion funnels for any time window |
| `create_campaign` | Set up new outreach campaigns |
| `lemleads_search` | Search lemlist lead database by ICP criteria (role, industry, company size, location) |
| `findEmail` | Locate verified email addresses (costs credits) |
| `verifyEmail` | Validate email authenticity (costs credits) |
| `linkedinEnrichment` | Enrich contact information via LinkedIn data (costs credits) |
| `findPhone` | Discover phone numbers for prospects (costs credits) |

Additional tools exist for lead import/management, campaign editing/pausing/deleting, and unsubscribe list management.
