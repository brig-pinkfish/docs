**Server path:** `/tomtom` | **Type:** Application (Remote MCP) | **PCID required:** Yes

TomTom official MCP server. Geocoding, reverse geocoding, search, POI discovery, routing, traffic, static and dynamic maps, EV routing, and data visualization. Tools are discovered at runtime using an existing connection.

## Tools

| Tool | Description |
| --- | --- |
| `tomtom-geocode` | Convert street addresses to coordinates |
| `tomtom-reverse-geocode` | Convert coordinates to street addresses |
| `tomtom-fuzzy-search` | Natural language search for addresses, POIs, and coordinates |
| `tomtom-poi-search` | Search for points of interest by name or category |
| `tomtom-nearby` | Find nearby points of interest |
| `tomtom-routing` | Calculate routes between two points |
| `tomtom-waypoint-routing` | Calculate routes with multiple waypoints |
| `tomtom-reachable-range` | Calculate reachable range from a point |
| `tomtom-traffic` | Get traffic incidents in a bounding box |
| `tomtom-static-map` | Generate static map images |
| `tomtom-dynamic-map` | Generate dynamic/interactive map data |

## Authentication

Requires a TomTom PinkConnect connection. Auth type: API Key (via `tomtom-api-key` header).
