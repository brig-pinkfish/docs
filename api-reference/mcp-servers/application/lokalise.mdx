Lokalise is exposed as two remote MCP server paths. Tools are discovered at runtime from the official Lokalise MCP server.

## Server paths

| Server path | Description |
| --- | --- |
| `/lokalise-project-management-mcp` | Manage projects, contributors, translation keys, and tasks |
| `/lokalise-software-development-mcp` | Download translation files and check translation status |

## Tools — Project Management

| Tool | Description |
| --- | --- |
| `lokalise_list_projects` | List all translation projects |
| `lokalise_create_keys` | Create new translation keys with default values and platform assignments |
| `lokalise_list_keys` | List translation keys in a project |
| `lokalise_list_translations` | List translations for a project |
| `lokalise_list_languages` | List languages configured in a project |
| `lokalise_list_usergroups` | List user groups |
| `lokalise_create_task` | Create a translation task |
| `lokalise_list_tasks` | List tasks in a project |

| Tool | Description |
