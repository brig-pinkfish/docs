## zoom_download_recording

Download a Zoom meeting recording file (video/audio). Use zoom_list_recordings first to get the meeting UUID. Returns either artifactUrl (for direct reference - do NOT fetch the URL content) or downloadUrl with streaming format (for efficient downloading via pf.mcp.downloadFile() or MCP server proxy). Supports MP4 (video), M4A (audio), and other recording formats.

**Parameters:**

| Parameter | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| `meetingUuid` | string | Yes | — | Meeting UUID from recordings (e.g., "VUe3+A+uSwC1Rb1Cxo/L4w=="). Use the "uuid" field from zoom_list_recordings response. |
