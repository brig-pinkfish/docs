## linear_download_file

Download a file from Linear's authenticated storage (uploads.linear.app). Use this to download images embedded in issue descriptions. The image URLs are found in markdown format like ![name](https://uploads.linear.app/...) in issue description fields. By default returns a downloadUrl for streaming. Set stream=false for small files (&lt;2MB) to get immediate base64 content.

**Parameters:**

| Parameter | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| `url` | string | Yes | — | The full URL to download (must be an https://uploads.linear.app/ URL). Found in issue descriptions as markdown image references. |
| `stream` | boolean | No | `true` | RECOMMENDED: true (default). Returns downloadUrl + auth headers for streaming. Set to false only for small files (&lt;2MB) requiring immediate base64 content. |
| `extractText` | boolean | No | `true` | Automatically extracts text from supported formats (PDF, DOCX, PPTX, XLSX) and includes it in the response. Set to false to skip extraction. |
