| `apply-style` | Apply a specified styleguide's rules to provided text and return styled content |
| `get-jasper-style-guides` | Retrieve the Jasper style guide configuration available to the workspace |
| `get-jasper-brand-voices` | Retrieve Jasper brand voices available to the workspace |
| `get-jasper-audiences` | Retrieve Jasper audiences available to the workspace |
| `get-jasper-agents` | Retrieve a list of available Jasper agents to perform marketing tasks |
