**Type:** Application (Remote MCP) | **PCID required:** Yes

Parallel Web Systems is exposed as two remote MCP server paths. Tools are discovered at runtime from the official Parallel Web Systems MCP servers.

## Server paths

| Server path | Description |
| --- | --- |
| `/parallel-web-systems-search-official-mcp` | AI-powered web search and research tools |
| `/parallel-web-systems-task-official-mcp` | Deep research tasks and data enrichment workflows |

## Tools — Search MCP

| Tool | Description |
| --- | --- |
| `web_search` | Search the web for information and retrieve relevant results |
| `web_fetch` | Extract and retrieve content from specific URLs |

## Tools — Task MCP

| Tool | Description |
| --- | --- |
| `create_deep_research_task` | Initiate a deep research task; returns details to view progress |
| `create_task_group` | Initiate a task group to enrich multiple items in parallel |
| `get_result` | Retrieve results of both deep research and task groups in an LLM-friendly format |
