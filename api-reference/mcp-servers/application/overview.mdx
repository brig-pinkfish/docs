**Type:** Application (proxy) | **PCID required:** Yes

Application servers are proxies to external integrations. Each server wraps a third-party API (Slack, Salesforce, Google Sheets, etc.) and exposes it as a set of MCP tools.

## How Application Servers Work

1. **Connect** — Set up a connection in the Pinkfish dashboard to get a PCID (PinkConnect connection ID)
2. **Discover** — Call `tools/list` on the server, or use [`capabilities_discover`](/api-reference/mcp-servers/embedded/pinkfish-sidekick#capabilities_discover) and [`capability_details`](/api-reference/mcp-servers/embedded/pinkfish-sidekick#capability_details) to find available tools
3. **Call** — Invoke any tool using the standard JSON-RPC 2.0 pattern, passing the PCID in the request

For full details on setting up connections and obtaining PCIDs, see the [Connections guide](/api-reference/platform/connections).

## Generic Call Pattern

Every application server follows the same calling pattern. Replace `<server-path>`, `<PCID>`, and `<tool_name>` with your values:

```bash
curl -s -X POST "https://mcp.pinkfish.ai/<server-path>" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <API_KEY>" \
  -d '{
    "jsonrpc": "2.0",
    "id": 1,
    "method": "tools/call",
    "params": {
      "name": "<tool_name>",
      "arguments": {
        "PCID": "<your-pcid>",
        ...
      }
    }
  }'
```

## Discovering Tools at Runtime

Tool schemas for application servers are provided dynamically by the proxy layer and may change as integrations update. Use runtime discovery instead of static documentation:

```bash
# List all tools on a server
curl -s -X POST "https://mcp.pinkfish.ai/<server-path>" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <API_KEY>" \
  -d '{
    "jsonrpc": "2.0",
    "id": 1,
    "method": "tools/list",
    "params": {}
  }'
```

Or use the discovery tools on the Pinkfish Workflow Building server (`/pinkfish-sidekick`):

```bash
# Find tools by describing what you want to do
curl -s -X POST "https://mcp.pinkfish.ai/pinkfish-sidekick" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <API_KEY>" \
  -d '{
    "jsonrpc": "2.0",
    "id": 1,
    "method": "tools/call",
    "params": {
      "name": "capability_details",
      "arguments": {
        "tool_names": ["slack_send_message", "slack_list_channels"]
      }
    }
  }'
```
