## web-application_deploy_web_app

Deploy pre-built HTML to a shareable URL.

WHEN TO USE: User explicitly wants a hosted/shareable link (e.g., "deploy this", "give me a URL", "host this app", "make this accessible").

NOT FOR: Creating HTML - that's done in code-block nodes using guidance from the system prompt. This tool only handles deployment.

REQUIRES: collectionId - filestorage collection to store the deployed HTML (use &#123;&#123;resource.collectionName&#125;&#125;).

DATA-DRIVEN APPS: If your HTML contains &#123;&#123;DATA_URL&#125;&#125; placeholder, provide sdk_file_id or filestorage_id to resolve it.

**Parameters:**

| Parameter | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| `html_content` | string | Yes | — | Required: The HTML code to deploy. Include &#123;&#123;DATA_URL&#125;&#125; placeholder if your app needs to fetch external data. |
| `sdk_file_id` | string | No | — | Supply data to your app: File ID from pf.files.writeFile() to resolve for &#123;&#123;DATA_URL&#125;&#125;. Format: automationId/runId/nodeId/filename. NOTE: The resolved S3 URL expires after approximately 15 minutes, so the deployed HTML should fetch the data immediately when loaded. |
| `filestorage_id` | string | No | — | Supply data to your app (preferred): File ID from filestorage MCP tool response to resolve for &#123;&#123;DATA_URL&#125;&#125;. Expiration depends on how the file was uploaded (can range from 1 minute to public/never). |
| `collectionId` | string | Yes | — | REQUIRED: Collection ID to store the deployed HTML file. In workflows, declare a filestorage collection resource in WORKFLOW_RESOURCES (e.g., myWebApps: &#123; type: "collection", collectionType: "filestorage" &#125;), bind it via workflow_bind_resources, then use &#123;&#123;resource.myWebApps&#125;&#125; here. |
| `output_filename` | string | No | `"web_app.html"` | Optional: Name for the output HTML file (default: "web_app.html") |
| `file_links_expire_in_days` | string | No | `7` | Number of days before the HTML URL expires (1-30), or "never" for a public URL with no expiration (default: 7) |
