## search_googlesearch

Perform a Google web search with advanced filtering options including geo-targeting, language, time range, file type, and device emulation.

**Parameters:**

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `q` | string | Yes | Search query |
| `gl` | string | No | Country code for geo-targeting (e.g. `us`, `uk`) |
| `hl` | string | No | Language code for results (e.g. `en`, `es`) |
| `location` | string | No | Location for localized results (e.g. `Austin, Texas`) |
| `tbs` | string | No | Time-based search filter (e.g. `qdr:d` for past day, `qdr:w` for past week) |
| `lr` | string | No | Restrict results to a specific language (e.g. `lang_en`) |
| `safe` | string | No | Safe search setting (`active` or `off`) |
| `filter` | string | No | Enable or disable duplicate content filtering (`0` or `1`) |
| `site` | string | No | Restrict results to a specific site (e.g. `reddit.com`) |
| `filetype` | string | No | Restrict results to a file type (e.g. `pdf`, `doc`) |
| `linksto` | string | No | Find pages linking to this URL |
| `related` | string | No | Find pages related to this URL |
| `start` | number | No | Result offset for pagination (default `0`) |
| `num` | number | No | Number of results to return (default `10`) |
| `device` | string | No | Device to emulate (`desktop`, `tablet`, `mobile`) |
| `no_cache` | boolean | No | Bypass cache and force a fresh search |

**Response fields:**

| Field | Type | Description |
| --- | --- | --- |
| `search_metadata` | object | Metadata about the search request |
| `search_parameters` | object | Echo of the search parameters used |
| `search_information` | object | Total results count and search time |
| `knowledge_graph` | object | Knowledge Graph panel data (if present) |
| `ai_overview` | object | AI-generated overview summary (if present) |
| `organic_results` | array | Search results with `position`, `title`, `link`, `snippet`, `date`, `source` |
| `related_questions` | array | "People also ask" questions |
| `related_searches` | array | Suggested related searches |
| `pagination` | object | Pagination info and next page tokens |

| `currency` | string | No | Currency code for prices (e.g. `USD`) |
| `sort_by` | enum | No | Sort: `1` (best), `2` (price), `3` (departure), `4` (arrival), `5` (duration), `6` (emissions) |

| `max_duration` | number | No | Maximum flight duration in minutes |
| `departure_token` | string | No | Token for selecting a specific departure flight |
