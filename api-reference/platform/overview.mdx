HTTP access to 1500+ tools via a single auth flow and JSON-RPC 2.0.

## Base URLs

| Service      | URL                             |
| ------------ | ------------------------------- |
| Platform API | https://app-api.app.pinkfish.ai |
| MCP Farm     | https://mcp.app.pinkfish.ai     |
| PinkConnect  | https://proxy.pinkfish.ai       |

## Quick Start

```bash
# 1. Exchange API key for runtime token
export PINKFISH_TOKEN=$(curl -s -X POST "https://app-api.app.pinkfish.ai/auth/token" \
  -H "X-Api-Key: YOUR_API_KEY" \
  -H "X-Selected-Org: YOUR_ORG_ID" \
  -H "Content-Type: application/json" | jq -r '.token')

# 2. Call a tool
curl -s -X POST "https://mcp.app.pinkfish.ai/web-search" \
  -H "Authorization: Bearer $PINKFISH_TOKEN" \
  -H "Content-Type: application/json" \
  -H "Accept: application/json" \
  -d '{"jsonrpc":"2.0","method":"tools/call","params":{"name":"search_googlesearch","arguments":{"query":"latest AI news"}},"id":1}'
```

## Example Script

Runnable script: authenticate, call web search, print result. Requires `curl` and `jq`.

```bash
#!/bin/bash
API_KEY="<YOUR_API_KEY>"
ORG_ID="<YOUR_ORG_ID>"
API_URL="https://app-api.app.pinkfish.ai"
MCP_URL="https://mcp.app.pinkfish.ai"

TOKEN_RESPONSE=$(curl -s -X POST "${API_URL}/auth/token" \
  -H "X-Api-Key: ${API_KEY}" -H "X-Selected-Org: ${ORG_ID}" -H "Content-Type: application/json")
PINKFISH_TOKEN=$(echo "$TOKEN_RESPONSE" | jq -r '.token')
[ "$PINKFISH_TOKEN" = "null" ] || [ -z "$PINKFISH_TOKEN" ] && { echo "Auth failed"; echo "$TOKEN_RESPONSE" | jq .; exit 1; }

RESULT=$(curl -s -X POST "${MCP_URL}/web-search" \
  -H "Authorization: Bearer ${PINKFISH_TOKEN}" -H "Content-Type: application/json" -H "Accept: application/json" \
  -d '{"jsonrpc":"2.0","method":"tools/call","params":{"name":"search_googlesearch","arguments":{"query":"latest AI news"}},"id":1}')
echo "$RESULT" | jq '.result.structuredContent // .result.content[0].text'
```

## Key Concepts

| Concept       | Description                                                       |
| ------------- | ----------------------------------------------------------------- |
| API Key       | Account credential. Exchange for runtime token.                   |
| Runtime Token | JWT from `/auth/token`. Bearer token for MCP requests.             |
| MCP Farm      | JSON-RPC 2.0 gateway over HTTPS.                                  |
| Server        | Tool collection (e.g. `gmail`, `web-search`).                     |
| Tool          | Callable action (e.g. `search_googlesearch`, `gmail_search_emails`). |
| PCID          | Connection ID. Required for OAuth integrations (Gmail, Slack).     |

## Prerequisites

- Pinkfish account
- API key from **Settings → API Keys → Create** (shown once; copy immediately)
- Organization ID from organization settings

<Note>New API keys may take up to 60 seconds to activate.</Note>
