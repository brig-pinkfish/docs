## Overview

The Platform API gives you programmatic access to the Pinkfish MCP Farm — **1500+ tools** behind a single auth flow and consistent protocol.

- **Built-in tools** — Web search, AI models, file storage, document processing
- **External integrations** — Gmail, Slack, Salesforce, Jira, 100+ others via OAuth
- **Discovery** — Find tools by natural language ("what can send an email?")
- **Code execution** — Sandboxed JavaScript to chain tool calls
- **Workflows** — Persistent automations with triggers and scheduling
- **Browser automation** — Natural language control of Playwright

## Key Concepts

| Concept | Description |
|---------|-------------|
| **API Key** | Account-level key (Settings → API Keys). Exchanged for a runtime token. |
| **Runtime Token** | JWT from `/auth/token`. Required for all MCP Farm calls. |
| **MCP Farm** | Unified gateway for all tools. JSON-RPC 2.0 over HTTPS. |
| **MCP Server** | Named collection of tools (e.g., `gmail`, `web-search`). |
| **Tool** | Single callable action (e.g., `gmail_search_emails`, `search_googlesearch`). |
| **PCID** | PinkConnect Connection ID for OAuth-based integrations. |

1. A Pinkfish account
2. An API key — create one in **Settings → API Keys → Create**

<Warning>
Copy the API key value immediately — it is shown only once. It can take up to 60 seconds for a new API key to activate.
</Warning>

## Quick Start

The full flow from authentication to calling a tool:

```bash
# 1. Exchange your API key for a runtime token
export PINKFISH_TOKEN=$(curl -s -X POST "https://app-api.app.pinkfish.ai/auth/token" \
  -H "X-Api-Key: YOUR_API_KEY" \
  -H "X-Selected-Org: YOUR_ORG_ID" \
  -H "Content-Type: application/json" | jq -r '.token')

# 2. Call a tool (example: web search)
curl -s -X POST "https://mcp.app.pinkfish.ai/web-search" \
  -H "Authorization: Bearer $PINKFISH_TOKEN" \
  -H "Content-Type: application/json" \
  -H "Accept: application/json" \
  -d '{
    "jsonrpc": "2.0",
    "method": "tools/call",
    "params": {
      "name": "search_googlesearch",
      "arguments": { "query": "latest AI news" }
    },
    "id": 1
  }'
```
