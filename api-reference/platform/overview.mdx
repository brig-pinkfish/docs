The Platform API provides HTTP access to the Pinkfish MCP Farm. Over 1500 tools are available behind a single authentication flow and consistent JSON-RPC 2.0 protocol.

## Base URLs

| Service      | URL                             |
| ------------ | ------------------------------- |
| Platform API | https://app-api.app.pinkfish.ai |
| MCP Farm     | https://mcp.app.pinkfish.ai     |
| PinkConnect  | https://proxy.pinkfish.ai       |

## Quick Start

```bash
# 1. Exchange API key for runtime token
export PINKFISH_TOKEN=$(curl -s -X POST "https://app-api.app.pinkfish.ai/auth/token" \
  -H "X-Api-Key: YOUR_API_KEY" \
  -H "X-Selected-Org: YOUR_ORG_ID" \
  -H "Content-Type: application/json" | jq -r '.token')

# 2. Call a tool
curl -s -X POST "https://mcp.app.pinkfish.ai/web-search" \
  -H "Authorization: Bearer $PINKFISH_TOKEN" \
  -H "Content-Type: application/json" \
  -H "Accept: application/json" \
  -d '{"jsonrpc":"2.0","method":"tools/call","params":{"name":"search_googlesearch","arguments":{"query":"latest AI news"}},"id":1}'
```

## Example Script: Auth + Web Search

A complete runnable script that exchanges your API key for a token, calls web search, and prints the result:

```bash
#!/bin/bash
API_KEY="<YOUR_API_KEY>"
ORG_ID="<YOUR_ORG_ID>"
API_URL="https://app-api.app.pinkfish.ai"
MCP_URL="https://mcp.app.pinkfish.ai"

# 1. Get token
TOKEN_RESPONSE=$(curl -s -X POST "${API_URL}/auth/token" \
  -H "X-Api-Key: ${API_KEY}" \
  -H "X-Selected-Org: ${ORG_ID}" \
  -H "Content-Type: application/json")
PINKFISH_TOKEN=$(echo "$TOKEN_RESPONSE" | jq -r '.token')
if [ "$PINKFISH_TOKEN" = "null" ] || [ -z "$PINKFISH_TOKEN" ]; then
  echo "Error: Failed to get token"; echo "$TOKEN_RESPONSE" | jq .; exit 1
fi

# 2. Call web search
RESULT=$(curl -s -X POST "${MCP_URL}/web-search" \
  -H "Authorization: Bearer ${PINKFISH_TOKEN}" \
  -H "Content-Type: application/json" \
  -H "Accept: application/json" \
  -d '{"jsonrpc":"2.0","method":"tools/call","params":{"name":"search_googlesearch","arguments":{"query":"latest AI news"}},"id":1}')

# 3. Print result (structuredContent or content[0].text)
echo "$RESULT" | jq '.result.structuredContent // .result.content[0].text'
```

<Note>Requires `curl` and `jq`.</Note>

## Key Concepts

| Concept       | Description                                                                      |
| ------------- | -------------------------------------------------------------------------------- |
| API Key       | Account-level credential from Settings. Exchanged for a runtime token.           |
| Runtime Token | JWT from /auth/token. Bearer token for all MCP Farm requests.                    |
| MCP Farm      | Unified gateway. JSON-RPC 2.0 over HTTPS.                                        |
| Server        | Named collection of tools such as gmail or web-search.                           |
| Tool          | Callable action such as gmail_search_emails or search_googlesearch.              |
| PCID          | PinkConnect Connection ID. Required for OAuth integrations like Gmail and Slack. |

## Prerequisites

- Pinkfish account
- API key from Settings, API Keys, Create

The API key is shown only once. Copy it immediately. New API keys can take up to 60 seconds to activate.
