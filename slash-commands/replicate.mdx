## What can you do with it?

The `/replicate` command enables you to run various AI models through the Replicate platform. You can generate images from text descriptions, synthesize speech from text, remove backgrounds from images, and leverage other AI capabilities using state-of-the-art models without managing infrastructure.

## How to use it?

### Basic Command Structure
```
/replicate [model] [input]
```

### Parameters

**Required:**

- `model` - The model to run (e.g., flux-schnell, speech-02-turbo)
- `input` - Input parameters specific to the model

**Optional:**

- `version` - Specific model version to use
- `fileLinksExpireInDays` - How long generated files remain accessible (1-30 days, defaults to 7)
- `num_outputs` - Number of outputs to generate (for image models)

### Response Format

The command returns:
```json
{
  "output": [
    {
      "url": "https://generated-file-url",
      "mimeType": "image/webp or audio/wav"
    }
  ],
  "model": "model used",
  "processing_time": "time in seconds"
}
```

## Examples

### Basic Usage
```
/replicate generate an image using flux-schnell
prompt: A modern office space with plants
aspect ratio: 16:9
```
Generates a quick image of an office space using the fast Flux model.

### Advanced Usage
```
/replicate text to speech using speech-02-turbo
text: Welcome to our automated customer service. How can I help you today?
voice: Young Woman
speed: normal
emotion: friendly
keep files for: 30 days
```
Generates friendly speech audio with a young woman's voice that remains accessible for 30 days.

### Specific Use Case
```
/replicate remove background using background-remover
image url: https://example.com/product-photo.jpg
output format: png
```
Removes the background from a product photo and returns a transparent PNG.

### Notes

**Recommended Models:**
- flux-schnell: Fast image generation
- flux-dev: High-quality image generation (slower)
- speech-02-turbo: Text-to-speech synthesis
- background-remover: Remove image backgrounds

**Supported Content Types:**
- Images: Generate, edit, and transform
- Audio: Generate speech and process audio
- Video: Not currently supported
