## What can you do with it?

Search for flight information including prices, schedules, airlines, and routes. Get real-time flight data with options for round trip, one-way, or multi-city searches.

## How to use it?

### Basic Command Structure
```
/flights
from: departure city or airport
to: arrival city or airport
departure date: when to leave
```

### Parameters
- **from**: Departure location (city name or IATA code)
- **to**: Arrival location (city name or IATA code)
- **departure date**: Date of departure (YYYY-MM-DD format)
- **return date** (optional): For round trips
- **trip type** (optional): Round trip, one way, or multi-city
- **class** (optional): Economy, premium economy, business, or first
- **passengers** (optional): Number of adults and children
- **stops** (optional): Nonstop, 1 stop or fewer, etc.

### Response Format
Returns comprehensive flight data including best flights, prices, durations, airlines, and carbon emissions information.

## Examples

### Basic Usage
Search for one-way flights:
```
/flights
from: New York
to: Los Angeles
departure date: 2024-03-15
```

### Advanced Usage
Search for round trip with preferences:
```
/flights
from: London
to: Tokyo
departure date: 2024-04-01
return date: 2024-04-15
class: business
stops: nonstop only
```

### Specific Use Case
Family vacation search:
```
/flights
from: Chicago
to: Orlando
departure date: 2024-07-20
return date: 2024-07-27
adults: 2
children: 2
max price: 2000
```

### Notes
- Automatically converts city names to IATA codes
- Provides carbon emissions data for eco-conscious travelers
- Shows price insights and typical price ranges
- Includes airline logos and flight amenities
- Flags frequently delayed flights
