The `/brightdata` command enables you to manage your BrightData proxy infrastructure and web scraping operations. Perfect for:

- Managing proxy zones
- Monitoring IP usage
- Configuring proxy settings
- Handling authentication
- Tracking billing and usage

## Basic Usage

Use the command to interact with BrightData:

```
/brightdata check account status
```

```
/brightdata create residential zone "my_scraping_zone" for country "us"
```

```
/brightdata list all zones
```

```

```

### Update Zone
```

```
