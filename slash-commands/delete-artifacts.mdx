## What can you do with it?

Delete artifact files that you downloaded or generated in previous steps. This is useful for cleaning up files in cases where you don't want them to persist on Pinkfish storage.

## How to use it?

### Basic Command Structure

```
/delete-artifacts
artifacts: which artifacts to delete
```

### Parameters

- **artifacts**: `All` or `specify filename`
- **include logs** (optional): Whether to include log files

### Response Format

Returns detailed results with success/failure counts and specific error messages for failed deletions.

## Examples

### Delete all

```
/delete-artifacts
artifacts: all
```

### Delete by name

```
/delete-artifacts
artifacts: report.pdf
```

### Delete files and logs

```
/delete-artifacts
artifacts: all
include logs: true
```

### Notes

- Log files (stdout.txt) are not deleted by default unless explicitly requested
- Deletion is permanent and cannot be undone
- Delete URLs expire after 15 minutes
