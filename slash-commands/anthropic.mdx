## What can you do with it?

The `/anthropic` command enables you to interact with Claude AI models for various tasks including complex reasoning and analysis, code generation and review, image understanding, research and writing, and creative tasks. This command provides access to state-of-the-art AI capabilities for both text and image processing.

## How to use it?

### Basic Command Structure
```
/anthropic [prompt] [optional-parameters]
```

### Parameters

**Required:**

- `prompt` - Your instructions or questions for Claude

**Optional:**

- `model` - Specific Claude model to use (defaults to claude-3-5-sonnet-20241022)

- `system` - System prompt to set custom behaviors or response styles

- `image` - Image file to analyze or process

### Response Format

The command returns:
```json
{
  "response": "Claude's generated response",
  "model": "model used for generation",
  "metadata": {
    "tokens": "number of tokens used",
    "processing_time": "time taken to process"
  }
}
```

## Examples

### Basic Usage
```
/anthropic
prompt: Explain quantum computing in simple terms
```
Asks Claude to provide a simple explanation of quantum computing.

### Advanced Usage
```
/anthropic
prompt: Review this code for bugs and suggest improvements
model: claude-opus-4-20250514
system: You are a helpful coding assistant
```
Uses the Opus model with a custom system prompt to review code with specific expertise.

### Specific Use Case
```
/anthropic
prompt: Describe what you see in this image and identify all objects
image: uploaded_image.jpg
```
Analyzes an uploaded image and provides detailed description of its contents.

### Notes 

Supported models include claude-opus-4-20250514, claude-sonnet-4-20250514, claude-3-5-sonnet-20241022 (default), claude-3-5-haiku-20241022, and legacy models. Images must be in JPEG, PNG, GIF, or WebP format.
