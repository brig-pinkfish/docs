## Why Use Translate?

The `/translate` command converts text from one language to another. Perfect for:

- Translating documents
- Converting messages
- Making content accessible in multiple languages
- Understanding foreign text
- Creating multilingual content

## Basic Usage

There are three ways to use the translate command:

1. Direct text translation:
```
/translate to French 
Hello, how are you?
```

2. File upload:
```
/translate to Spanish
document.txt
```

3. File URL:
```
Translate the following document to Hindi:
https://example.com/documents/report.pdf
```

## Supported Formats

- Text files (.txt)
- PDF (.pdf)
- Wav (.wav)
- Mp3 (.mp3)

## Tips
- For best results, ensure source text is clear and well-formatted
- Specify the target language for more accurate translations
- For technical content, include context when possible
- Consider regional language variations (e.g., European Spanish vs. Latin American Spanish)
