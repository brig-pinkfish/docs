The `/fireflies` command enables you to access and manage meeting transcripts, notes, and conversation analytics. Perfect for:

- Accessing meeting transcripts
- Searching conversations
- Managing action items
- Analyzing topics
- Creating notes

## Basic Usage

Use the command to interact with Fireflies:

```
/fireflies get transcripts from last 10 meetings
```

```
/fireflies search for "user engagement" across all meetings
```

```
/fireflies show details for transcript ff_tr_123456789
```

## GraphQL Queries

The Fireflies API uses GraphQL. Common queries include:

### Basic Transcript List
```graphql
query GetTranscripts($limit: Int, $offset: Int) {
  transcripts(limit: $limit, offset: $offset) {
    title
    date
    transcript_id
    duration_seconds
  }
}
```

### Detailed Transcript
```graphql
query GetTranscriptDetails($transcript_id: ID!) {
  transcript(transcript_id: $transcript_id) {
    sentences { speaker_name timestamp text }
    topics { title start_time end_time }
    action_items { text assignee { name } due_date }
  }
}
```

## Response Data

### Transcript Info
- Title and date

- Duration in seconds

### Sentence Data
