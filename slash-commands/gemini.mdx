## What can you do with it?

Use Gemini for tasks involving text, images, audio, and video. Perfect for complex reasoning, multimodal analysis, content generation, and real-time interactions.

## How to use it?

### Basic Command Structure
```
/gemini
prompt: your instructions or questions for Gemini
```

### Parameters
- **prompt**: Your instructions or questions for Gemini
- **model** (optional): Specific Gemini model to use (defaults to gemini-2.0-flash)
- **system prompt** (optional): Override the default system prompt
- **files** (optional): File URLs to include (supports audio, images, videos, and text)

### Response Format
Returns Gemini's response in the requested format (JSON by default, or plaintext/markdown/HTML if specified).

## Examples

### Basic Usage
Get a simple response from Gemini:
```
/gemini
prompt: Explain the concept of machine learning
```

### Advanced Usage
Analyze multimedia content:
```
/gemini
prompt: Describe what's happening in this video and transcribe any dialogue
files: [video_file_url]
model: gemini-2.5-pro
```

### Specific Use Case
Generate text-to-speech audio:
```
/gemini
prompt: Convert this text to natural speech: "Welcome to our presentation"
model: gemini-2.5-pro-tts
```

### Notes
Key models include:
- gemini-2.0-flash: Fast, general tasks (default)
- gemini-2.5-pro: Enhanced reasoning and multimodal understanding
- gemini-2.5-flash-tts: Text-to-speech generation
- gemini-2.0-flash-preview-image-generation: Image generation
- gemini-1.5-pro: Complex reasoning with long context
- gemini-2.0-flash-live-001: Real-time voice and video interactions

## Why Use Gemini?

The `/gemini` command gives you access to Google's Gemini AI models. Perfect for:

- Natural language processing
- Content generation
- Data analysis
- Problem-solving
- Creative writing
- Image analysis
- Speech to text

## Basic Usage

Simply use the command followed by your query:

```
/gemini Analyze market trends in the renewable energy sector
with file market_data.csv
```

You can also provide multiple documents:

```
/gemini Compare Q4 performance across regions
with file q4_north.csv
with file q4_south.csv
```

Or no documents at all:

```
/gemini Give me a list of the top 100 companies in the world
```

## Model Selection

You can specify which Gemini model you want to use. 
See: https://ai.google.dev/gemini-api/docs/models/gemini

```
/gemini use gemini-1.5-flash-002
Analyze this technical documentation
with file api_specs.pdf
```

## Supported File Types

### Text Documents
• PDF (.pdf)
• Microsoft Word (.docx)
• Plain Text (.txt)
• Rich Text Format (.rtf)
• HTML (.html)
• OpenDocument Text (.odt)
• EPUB (.epub)

### Structured Data
• CSV (.csv)
• JSON (.json)

### Images
• JPEG (.jpg, .jpeg)
• PNG (.png)
• GIF (.gif)
• WebP (.webp)
