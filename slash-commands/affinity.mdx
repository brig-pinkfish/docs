The `/affinity` command enables you to manage and access your Affinity CRM data. Perfect for:

- Viewing lists and contacts
- Retrieving field values
- Managing CRM data
- Accessing person information
- Working with custom fields

## Basic Usage

Use the command to interact with Affinity CRM:

```
/affinity get all my lists
```

```
/affinity get field values for person ID 12345
```

```
/affinity show list details for "My List of People"
```

## Key Features

### List Management
- View all accessible lists
- Check list details
- See list owners
- Monitor list sizes
- Access public/private lists

### Field Values
- Retrieve person field values
- Access custom field data
- View location information
- Check field history
- Monitor field updates

### Data Access
- Query person information
- Access CRM records
- View entity details
- Check timestamps

## Example Commands

### Get All Lists
```
/affinity show all my accessible lists
```

### Get Person Field Values
```
/affinity get field values for person 38706
```

### View List Information
```
/affinity show details for list ID 450
```

### Check Custom Fields
```
/affinity get location field for person ID 38706
```

## List Types

Common list types:
- **Type 0**: People lists
- **Type 1**: Company lists
- **Type 2**: Opportunity lists

## Field Value Structure

Field values include:
- **Location fields**: City, state, country
- **Custom fields**: User-defined data
- **System fields**: Created/updated timestamps
- **Entity associations**: Links to lists and entries

## Tips
- Use person IDs for specific field queries
- Lists show owner and size information
- Field values include full location data
- All timestamps are in ISO format
- Check list type before querying
