The `/microsoft-powerbi` command enables you to create, manage, and share data visualizations, dashboards, and reports through Microsoft Power BI. Perfect for:

- Managing workspaces and reports
- Creating and sharing dashboards
- Dataset management and refresh
- Report embedding and export
- Data visualization automation

## Basic Usage

Use the command to work with Power BI:

```
/microsoft-powerbi get all workspaces and their reports
```

```
/microsoft-powerbi refresh dataset "Sales Data 2024" in workspace
```

```
/microsoft-powerbi export report "Quarterly Performance" to PDF
```

## Workspace Operations

### Get Workspaces
```javascript
const url = POWERBI_URL + "myorg/groups";
```

### Create Workspace
```javascript
{
  "name": "Financial Analytics 2025"
}
```

### Workspace Response
```javascript
{
  "value": [
    {
      "id": "3d9b93c6-7b6d-4801-a491-1738910904fd",
      "name": "Marketing Analytics",
      "isReadOnly": false,
      "isOnDedicatedCapacity": true,
      "capacityId": "8a5e290f-6b73-4681-82f5-abcd123456",
      "type": "Workspace"
    }
  ]
}
```

## Report Operations

### Get Reports in Workspace
```javascript
const url = POWERBI_URL + `myorg/groups/${workspaceId}/reports`;
```

### Get Report Details
```javascript
const url = POWERBI_URL + `myorg/groups/${workspaceId}/reports/${reportId}`;
```

### Clone Report
```javascript
{
  "name": "Quarterly Performance - Marketing Copy",
  "targetModelId": "cfafbeb1-8037-4d0c-896e-a46fb27ff229",
  "targetWorkspaceId": "9f41b6a8-0241-4301-9731-0dca082b0c0e"
}
```

### Export Report
```javascript
const url = POWERBI_URL + `myorg/groups/${workspaceId}/reports/${reportId}/Export?format=PDF`;
```

## Dataset Management

### Get Datasets
```javascript
const url = POWERBI_URL + `myorg/groups/${workspaceId}/datasets`;
```

### Refresh Dataset
```javascript
// POST request with empty payload
const url = POWERBI_URL + `myorg/groups/${workspaceId}/datasets/${datasetId}/refreshes`;
```

### Get Refresh History
```javascript
const url = POWERBI_URL + `myorg/groups/${workspaceId}/datasets/${datasetId}/refreshes`;
```

### Dataset Response
```javascript

  "value": [
```

## Embedding and Tokens

### Generate Embed Token
```javascript
{
  "accessLevel": "View",
  "allowSaveAs": false,
  "datasetId": "cfafbeb1-8037-4d0c-896e-a46fb27ff229"
}
```

### Token Response
```javascript
{
  "token": "H4sIAAAAAAAEAB2Ux6K7CBKE3z...",
  "tokenId": "a1b2c3d4-5e6f-7g8h-9i0j-1k2l3m4n5o6p",
  "expiration": "2025-03-10T12:30:45Z"
}
```

### Access Levels
- **View**: Read-only access
- **Edit**: Full editing capabilities
- **Create**: Can create new content

## Report Structure

### Report Properties
- **id**: Unique report identifier
- **name**: Report display name
- **webUrl**: Direct access URL
- **embedUrl**: Embedding URL
- **datasetId**: Associated dataset
- **createdDateTime**: Creation timestamp

### Report Pages
```javascript
const url = POWERBI_URL + `myorg/groups/${workspaceId}/reports/${reportId}/pages`;
```

### Page Response
```javascript
{
  "value": [

      "order": 0
    },
    {
      "name": "ReportSection2",

```

## Refresh Operations

### Refresh Status Types
- **Unknown**: Status not determined
- **Completed**: Successfully completed
- **Failed**: Refresh failed
- **InProgress**: Currently refreshing
- **NotStarted**: Queued but not started

### Refresh History Response
```javascript
{
  "value": [
    {
      "requestId": "bf17ac4e-93e3-4e8c-a2e6-5f8fc833e1bc",

      "endTime": "2025-03-09T08:32:15.456Z",
      "status": "Completed",
      "serviceExceptionJson": null
    }
  ]
}
```

## Export Formats

### Supported Formats
- **PDF**: Portable Document Format
- **PPTX**: PowerPoint presentation
- **PNG**: Image format
- **XLSX**: Excel spreadsheet
- **CSV**: Comma-separated values

### Export Response
```javascript
{
  "status": 200,
  "headers": {
    "Content-Type": "application/pdf",
    "Content-Disposition": "attachment; filename=Quarterly Performance.pdf"
  },
  "output": {
    "file": "binary-file-data",
    "message": "Report exported successfully."
  }
}
```
