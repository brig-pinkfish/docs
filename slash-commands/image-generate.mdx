## What can you do with it?

The `/image-generate` command enables you to create unique images from text descriptions using AI. You can generate artwork, product mockups, marketing visuals, concept designs, and illustrations by simply describing what you want to see, with control over style, composition, and quality.

## How to use it?

### Basic Command Structure
```
/image-generate [description]
```

### Parameters

**Required:**

- `description` - Text description of the image you want to generate

**Optional:**

- `aspect_ratio` - Image dimensions: 1:1, 16:9, 9:16, 4:3, or 3:4 (defaults to 1:1)
- `style` - Visual style preferences (realistic, artistic, cartoon, etc.)
- `quality` - Output quality: standard or high (defaults to standard)
- `num_outputs` - Number of variations to generate: 1-4 (defaults to 1)

### Response Format

The command returns:
```json
{
  "image_url": "URL to the generated image",
  "output": ["array of generated image URLs"],
  "metrics": {
    "generation_time": "time in seconds",
    "image_count": "number of images generated"
  },
  "prompt_used": "the processed prompt sent to the model"
}
```

## Examples

### Basic Usage
```
/image-generate
description: a modern office space with plants and natural lighting
```
Generates a single image of an office space based on the description.

### Advanced Usage
```
/image-generate
description: futuristic city skyline at sunset with flying vehicles
aspect_ratio: 16:9
style: cyberpunk aesthetic with neon lights
quality: high
num_outputs: 3
```
Creates three high-quality variations of a futuristic cityscape in widescreen format.

### Specific Use Case
```
/image-generate
description: minimalist logo design for a tech startup named CloudSync
aspect_ratio: 1:1
style: clean, professional, blue and white color scheme
```
Generates a logo design concept for branding purposes with specific style requirements.

### Notes

**Limitations:**
- Cannot generate images of specific real people
- Avoids creating harmful or inappropriate content
- Results may vary based on prompt complexity
- Generation typically takes 10-30 seconds
