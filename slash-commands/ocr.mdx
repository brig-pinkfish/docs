## Why Use OCR?

The `/ocr` command helps you extract text from images quickly and easily. It's perfect for converting text in:

- Screenshots
- Photos of documents
- Images with text content
- Handwritten notes

## Basic Usage

Simply use the command followed by your image:

```
/ocr [uploaded image or URL to an image]
```
Or specify the output format:
```
/ocr [uploaded image or URL to an image] 
extract only the data from charts and tables
output as a CSV
```

## Supported Formats

- JPG/JPEG
- PNG

## Note

For PDF documents, please use the `/pdf-to-text` command.
