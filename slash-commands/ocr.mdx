## Why Use OCR?

The `/ocr` command helps you extract text from images quickly and easily. It's perfect for converting text in:
- Screenshots
- Photos of documents
- Images with text content
- Handwritten notes

## Basic Usage

Simply use the command followed by your image:

```
/ocr [uploaded image or provide URL]
```

## Supported Formats
- JPG/JPEG
- PNG

## Note
For PDF documents, please use the `/text-to-pdf` command instead.
