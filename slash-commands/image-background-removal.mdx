## What can you do with it?

The `/image-background-removal` command enables you to automatically remove backgrounds from images using AI. You can isolate subjects from photos, create transparent backgrounds for product images, prepare images for compositing, and clean up photos for professional use without manual editing.

## How to use it?

### Basic Command Structure
```
/image-background-removal [image]
```

### Parameters

**Required:**

- `image` - The image to process (URL or uploaded file)

**Optional:**

- `output_format` - Format for the result: png or webp (defaults to png for transparency)
- `edge_refinement` - Refine edges for cleaner cutouts (true/false)

### Response Format

The command returns:
```json
{
    "output": [
        {
            "url": "https://.... ",
            "mimeType": "image/png"
        }
    ]
}
```

## Examples

### Basic Usage
```
/image-background-removal
image: portrait-photo.jpg
```
Removes the background from a portrait photo, leaving only the subject.

### Advanced Usage
```
/image-background-removal
image: product-photo.png
output_format: webp
edge_refinement: true
```
Processes a product photo with refined edges and outputs in WebP format for web optimization.

### Specific Use Case
```
/image-background-removal
image: team-member-headshot.jpg
output_format: png
```
Creates a professional headshot with transparent background for use in presentations or websites.

### Notes

**Supported Image Formats:**
- JPEG/JPG
- PNG
- WebP
- BMP

**Output Considerations:**
- PNG format preserves transparency
- WebP offers smaller file sizes with transparency
- Processing typically takes 5-15 seconds
- Works best with clear subject-background contrast
