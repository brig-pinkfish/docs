## What can you do with it?

The `/3d-app` command enables you to create complete, interactive 3D web applications using Babylon.js. You can build product viewers, configurators, educational models, architectural walkthroughs, and interactive 3D experiences that run directly in the browser with camera controls, lighting, materials, and animations.

## How to use it?

### Basic Command Structure
```
/3d-app [scene-description]
```

### Parameters

**Required:**

- `scene-description` - Detailed description of the 3D scene you want to create including objects, lighting, camera setup, and interactions

**Optional:**

- `For model use file: [file-url]` - URL to a GLB/GLTF 3D model file to include in the scene
- `Camera type: [camera-type]` - Specify camera type (ArcRotate, Free, Follow, Universal)
- `Lighting: [lighting-setup]` - Custom lighting configuration
- `Materials: [material-properties]` - Specific material and texture requirements
- `Animations: [animation-details]` - Animation and interaction specifications

### Response Format

The command returns:
```html
<!DOCTYPE html>
<html>
<head>
    <title>Babylon.js 3D Application</title>
    <style>[CSS styles]</style>
</head>
<body>
    <canvas id="renderCanvas"></canvas>
    <script src="https://cdn.babylonjs.com/babylon.js"></script>
    <script>[JavaScript 3D scene code]</script>
</body>
</html>
```

## Examples

### Basic Usage
```
/3d-app create a simple 3D scene with a rotating sphere
```
Creates a basic 3D scene with a sphere that rotates automatically, including camera controls and basic lighting.

### Advanced Usage
```
/3d-app create an interactive product viewer with multiple objects
For model use file: https://bjs-playground.s3.us-east-1.amazonaws.com/sneakers.glb
Camera type: ArcRotate
Lighting: Point lights with color changing effects
Materials: Metallic and glass materials
Animations: Smooth rotation and hover effects
```
Creates a comprehensive product viewer with custom models, advanced lighting, multiple materials, and interactive animations.

### Specific Use Case
```
/3d-app create an architectural walkthrough of a simple room
Camera type: Free
Lighting: Realistic indoor lighting with shadows
Materials: Wood floors, painted walls, glass windows
Animations: Smooth camera movement through the space
```
Creates a first-person architectural walkthrough with realistic materials and lighting for interior visualization.
