## What can you do with it?

Search the web using Google's search engine with advanced filtering options. Access organic results, knowledge graphs, related questions, and more with localization and filtering capabilities.

## How to use it?

### Basic Command Structure
```
/google-search
query: what you want to search for
```

### Parameters
- **query**: Your search terms (required)
- **location** (optional): City-level location for localized results
- **language** (optional): Language code (en, es, fr, etc.)
- **country** (optional): Country code (us, uk, fr, etc.)
- **search type** (optional): Regular, images, videos, news, shopping, etc.
- **safe search** (optional): Filter adult content
- **results per page** (optional): Number of results to return

### Response Format
Returns comprehensive search results including organic results, knowledge graph data, related questions, and pagination information.

## Examples

### Basic Usage
Simple web search:
```
/google-search
query: best coffee shops in Seattle
```

### Advanced Usage
Localized search with filters:
```
/google-search
query: latest technology news
location: San Francisco
language: en
country: us
search type: news
```

### Specific Use Case
Product research:
```
/google-search
query: laptop reviews 2024
search type: regular
safe search: active
results per page: 20
```

### Notes
- Supports multiple search types: regular, images, videos, news, shopping, patents
- Provides knowledge graph information when available
- Includes related questions and searches
- Offers pagination for browsing multiple result pages
- Respects localization settings for relevant regional results
