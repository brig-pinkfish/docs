### id
firecrawl

### skill
Firecrawl

### description
With Firecrawl, retrieve a map of the URLs given a starting URL.

### tags
- web crawl mapping

### prompt
```javascript
<firecrawl_skill>
This is an internal skill. Use:
SCRAPER_FC_URL = INTERNAL_SKILL_URL + 'scraperfc/' //POST to this URL and nothing else. *Mandatory*

Example Postbody
const POST_BODY = {
    "urls": ["https://example.com", "https://another-example.com"],
     // === OUTPUT FORMAT OPTIONS ===
    "formats": [
        "markdown",           // IMPORTANT: By default: only include this option, other options must be excluded
    ],
    // === BROWSER ACTIONS ===
    // This is default value
    "actions": [
        {
            "type": "screenshot",          
            "fullPage": true
        },
        {
            "type": "scroll",
            "selector": "body",
            "direction": "down",          // "up" or "down"
        }
    ],
    "waitFor": 2000,
    "includeTags": ["div", "p", "h1", "h2"],
    "excludeTags": ["script", "style", "noscript"],
    "onlyMainContent": true,
    "removeBase64Images": true,
    "file_links_expire_in_days": 7
}

urls: Array of URLs to scrape (required)
formats: Output formats for the scraped content. Available formats:
    [
        "markdown",             // Clean markdown content.
        "html",                 // Processed HTML
        "rawHtml",              // Raw HTML source
        "links",                // Extracted links array
        "screenshot",           // Base64 screenshot data
        "screenshot@fullPage",  // Full page screenshot
        "json",                 // Structured data extraction
        "changeTracking"        // Change detection data
    ]
actions: Browser actions to perform before scraping ("wait", "click", "scroll", "screenshot"). If there is screenshot action then remove "screenshot@fullPage" in "formats" if any.
waitFor: Time to wait in milliseconds before scraping (default: 1000)
includeTags: HTML tags to include in the output
excludeTags: HTML tags to exclude from the output
onlyMainContent: Boolean to extract only the main content
removeBase64Images: Boolean to remove base64-encoded images
file_links_expire_in_days: Number of days until file links expire

Sample response:
{
    "https://www.google.com/": {
        "success": true,
        "data": {
            "markdown": "markdown here... ",
            "metadata": {
                "language": "en",
                "title": "Google",
                "referrer": "origin",
                "scrapeId": "2c29af85-dc7e-4301-b80a-c4da8bf178cb",
                "sourceURL": "https://www.google.com/",
                "url": "https://www.google.com/",
                "statusCode": 200
            }
        },
        "file_urls": []
    }
}

</firecrawl_skill>
```
