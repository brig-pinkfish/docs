The `/klaviyo` command enables you to manage email campaigns and subscriber lists through Klaviyo's marketing platform. Perfect for:

- Creating and managing email campaigns
- Building subscriber lists
- Managing email marketing automation
- Tracking campaign performance
- Subscriber management

## Basic Usage

Use the command to work with Klaviyo:

```
/klaviyo create "Winter Sale Campaign" for list 123456
```

```
/klaviyo add subscriber "john.doe@example.com" to list "New Subscribers"
```

```
/klaviyo send campaign "campaign_123456" at scheduled time
```

## Campaign Operations

### Create Campaign
```javascript
{
  "name": "Winter Sale Campaign",
  "list_id": "123456",
  "template_id": "654321",
  "subject": "Winter Sale is Here!",
  "from_email": "sales@company.com",
  "from_name": "Company Sales",
  "content_type": "html"
}
```

### Get Campaigns
```javascript
const url = KLAVIYO_URL + "campaigns";
```

### Send Campaign
```javascript
{
  "send_time": "2025-01-15T12:00:00Z"
}
```

## List Operations

### Create List
```javascript
{
  "name": "New Subscribers",
  "list_type": "list"
}
```

### Add Subscriber to List
```javascript
{
  "profiles": [
    {
      "email": "john.doe@example.com",
      "phone_number": "+1234567890",
      "first_name": "John",
      "last_name": "Doe"
    }
  ]
}
```

### Remove Subscriber
```javascript
{
  "emails": [
    "john.doe@example.com"
  ]
}
```

## Campaign Structure

### Campaign Properties
- **id**: Unique campaign identifier
- **name**: Campaign display name
- **status**: Current campaign status
- **created**: Creation timestamp
- **list_id**: Target audience list
- **template_id**: Email template

### Campaign Status
- **draft**: Being created/edited
- **sending**: Currently sending
- **sent**: Completed send
- **scheduled**: Scheduled for future
- **paused**: Temporarily stopped

## List Management

### List Properties
- **list_id**: Unique list identifier
- **name**: List display name
- **created**: Creation timestamp
- **list_type**: Type of list
- **subscriber_count**: Number of subscribers

### Subscriber Properties
- **email**: Primary email address
- **phone_number**: Contact phone
- **first_name**: Given name
- **last_name**: Family name
- **status**: Subscription status
