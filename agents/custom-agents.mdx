## Custom Agents

Create AI agents tailored to your specific business needs. Define custom instructions, connect applications, and attach workflows for specialized processes.

### Key Features

- **Custom Instructions**: Define your agent's role and behavior
- **App Integrations**: Connect to 200+ applications
- **Workflow Orchestration**: Attach existing workflows
- **Team Sharing**: Share agents with colleagues

### Common Use Cases

- **Employee Onboarding**: Guide new colleagues through setup
- **RFP Management**: Help teams fill out proposals
- **Customer Support**: Provide first-line support
- **IT Service Management**: Handle tickets and user provisioning
- **HR Operations**: Manage employee requests and policies
