## Universal Agent

The Universal Agent is your AI assistant for daily work, designed to bridge across all your applications and handle tasks seamlessly. It comes pre-built with access to all your connected apps and the full library of slash commands.

### Key Features

- **Cross-App Integration**: Works across all your connected applications as a bridge
- **Full Command Access**: Uses all 200+ slash commands for enhanced capabilities
- **Ready-to-Use**: No setup required - start working immediately
- **Daily Work Focus**: Designed for your everyday business tasks

### How It Works

The Universal Agent acts as your bridge between applications, enabling workflows like:
- Reading Zendesk tickets and creating corresponding Jira issues
- Processing emails and updating CRM records
- Extracting data from documents and adding to spreadsheets
- Coordinating tasks across multiple platforms

### Enhanced Capabilities

Beyond app integration, the Universal Agent has access to all slash commands:
- **Image Generation**: Create images, remove backgrounds, process visuals
- **Document Processing**: Extract text, fill forms, convert files
- **Data Analysis**: Structure JSON, analyze CSV files, generate reports
- **Communication**: Send formatted emails, create presentations
- **Web Research**: Search information, scrape websites, gather data

### Common Use Cases

- **Cross-Platform Workflows**: Sync data between different tools
- **Document Management**: Process and organize files across systems
- **Communication Automation**: Draft emails, schedule meetings, send updates
- **Data Processing**: Extract, transform, and load information
- **Creative Tasks**: Generate images, create presentations, design content

### Getting Started

1. The Universal Agent is automatically available in your workspace
2. Start a conversation and describe what you need help with
3. It will work across your connected apps and use appropriate commands
4. No configuration needed - just start working

*Your AI assistant is ready to help with your daily work across all your apps!*
