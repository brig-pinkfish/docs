## Hello World with Pinkfish

Let's get started with the simplest possible example.

### First: Write "Output Hello World"

Type `output hello world` in the chat entry.

### Next: Run Your Automation

When you get a code block back, click "Run"

### Results

Hopefull you'll now see "Hello World" in your results box.

Congratulations, you've just run your first step.
Happy automating!
