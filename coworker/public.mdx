Coworker can be exposed publicly at `/coworker/public`. Any agent you've released with a **public channel** can be reached from this URL without requiring a Pinkfish account.

## Use cases

- Customer-facing bots (support, onboarding, product questions).
- Demos that live at a shareable link.
- Landing pages where visitors can ask an agent a question before signing up.

## How to enable

1. Build an agent in [Agent Builder](/agents/agent-builder).
2. On the **Channels** tab, enable the **Public / External URL** channel.
3. Release the agent. The public URL is shown on the Channels tab after release.
